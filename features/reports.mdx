The Reports feature provides a comprehensive view of all your transactions with powerful filtering capabilities and visual analytics. Access reports through the `/reports` route.

## Transaction Reports Overview

The transaction reports page displays all your financial data in a clean, organized table format with the following information:

- Transaction date
- Amount
- Type (Income or Expense)
- Description
- Category
- Creation timestamp
- Action buttons (Edit/Delete)

## Viewing Transactions

When you navigate to the reports page, all transactions are loaded automatically:

```typescript
ngOnInit(): void {
  this.transactions$ = this.transactionService.getAllTransactions().pipe(
    map(res => {
      this.allTransactions = res.data;
      return res.data;
    })
  );
}
```

The transactions are displayed using Angular's async pipe for efficient change detection and memory management.

## Filtering by Date Range

The reports page includes a powerful date range filter that allows you to narrow down transactions to specific time periods.

<Steps>
  <Step title="Set Start Date">
    Select a start date using the "From" date picker. This sets the beginning of your date range.

    ```html
    <input 
      type="date" 
      [(ngModel)]="startDate"
      class="px-2 py-1 text-sm border border-gray-300 rounded"
    >
    ```
  </Step>

  <Step title="Set End Date">
    Select an end date using the "To" date picker. This sets the end of your date range.

    ```html
    <input 
      type="date" 
      [(ngModel)]="endDate"
      class="px-2 py-1 text-sm border border-gray-300 rounded"
    >
    ```
  </Step>

  <Step title="Apply Filter">
    Click the "Filter" button to apply your date range:

    ```typescript
    filterByDate(): void {
      if (!this.startDate && !this.endDate) {
        this.transactions$ = this.transactionService.getAllTransactions()
          .pipe(map(res => res.data));
        return;
      }

      const start = this.startDate ? new Date(this.startDate) : new Date('1900-01-01');
      const end = this.endDate ? new Date(this.endDate) : new Date('2100-12-31');

      const filtered = this.allTransactions.filter(transaction => {
        const transactionDate = new Date(transaction.date);
        return transactionDate >= start && transactionDate <= end;
      });

      this.transactions$ = new Observable(observer => {
        observer.next(filtered);
        observer.complete();
      });
    }
    ```
  </Step>

  <Step title="Clear Filters">
    Click the "Clear" button to remove all filters and show all transactions:

    ```typescript
    clearFilters(): void {
      this.startDate = '';
      this.endDate = '';
      this.transactions$ = this.transactionService.getAllTransactions()
        .pipe(map(res => res.data));
    }
    ```
  </Step>
</Steps>

<Note>
  If you specify only a start date, all transactions from that date forward are shown. If you specify only an end date, all transactions up to that date are shown.
</Note>

## Transaction Table Layout

The report table displays transaction data with the following columns:

| Column | Description | Format |
|--------|-------------|--------|
| Date | Transaction date | MMM dd, yyyy (e.g., "Mar 05, 2026") |
| Amount | Transaction amount | $X,XXX.XX (two decimal places) |
| Type | Income or Expense | Colored badge (green for Income, red for Expense) |
| Description | Optional description | Text (truncated if too long) |
| Category | Category name | Badge with category name snapshot |
| Activity | Creation timestamp | Short date format |
| Actions | Edit/Delete buttons | Icon buttons |

### Type Badges

Transaction types are displayed with color-coded badges:

```html
<span
  class="px-2 py-0.5 inline-flex text-xs font-medium rounded"
  [class.bg-emerald-100]="transaction.type === 'Income'"
  [class.text-emerald-700]="transaction.type === 'Income'"
  [class.bg-rose-100]="transaction.type === 'Expense'"
  [class.text-rose-700]="transaction.type === 'Expense'">
  {{ transaction.type }}
</span>
```

- **Income**: Green/emerald background with dark green text
- **Expense**: Red/rose background with dark red text

## Visual Analytics with Charts

For a visual representation of your financial data, MyFinance UI includes chart-based reports accessible via the `/chart` route.

### Monthly Report Generation

The report chart component fetches monthly aggregated data for both income and expenses:

```typescript
private loadReportData(): void {
  forkJoin({
    income: this.transactionService.getMonthlyReport('Income', this.currentYear - 1),
    expense: this.transactionService.getMonthlyReport('Expense', this.currentYear - 1)
  }).subscribe({
    next: ({ income, expense }) => {
      const incomeData = this.mapReportToChartData(income.data);
      const expenseData = this.mapReportToChartData(expense.data);

      this.chartOptions.series = [
        { type: 'area', name: 'Income', color: '#5ccb5f', data: incomeData },
        { type: 'area', name: 'Expense', color: '#f00', data: expenseData }
      ];

      this.updateFlag = true;
    }
  });
}
```

### Highcharts Integration

The chart is powered by Highcharts and displays data as an area chart:

```typescript
chartOptions: Highcharts.Options = {
  chart: { type: 'area' },

    title: { text: 'Month' }
  },

  tooltip: { shared: true, valuePrefix: '$' },
  series: [
    { type: 'area', name: 'Income', color: '#5ccb5f', data: [] },
    { type: 'area', name: 'Expense', color: '#f00', data: [] }
  ]
};
```

### Report Data Mapping

Monthly report data is mapped to chart-compatible format:

```typescript
private mapReportToChartData(reports: Report[]): number[] {
  const monthlyData = new Array(12).fill(0);
  
  reports.forEach(report => {
    if (report.month >= 1 && report.month <= 12) {
      monthlyData[report.month - 1] = report.total;
    }
  });

  return monthlyData;
}
```

This ensures all 12 months are represented, even if some months have no transactions (showing $0).

## API Integration

The reports feature uses these `TransactionService` methods:

```typescript
// Get all transactions
getAllTransactions(): Observable<ApiResponse<Transaction[]>> {
  return this.http.get<ApiResponse<Transaction[]>>(this.apiUrl);
}

// Get monthly report by type and year
getMonthlyReport(type: 'Income' | 'Expense', year: number): Observable<ApiResponse<Report[]>> {
  return this.http.get<ApiResponse<Report[]>>(`${this.apiUrl}/report?type=${type}&year=${year}`);
}
```
