<div className="mt-16 mb-16 max-w-5xl mx-auto px-6">
  <h2 className="text-3xl font-bold text-gray-900 dark:text-white mb-4">Quick Start</h2>
  <p className="text-base text-gray-600 dark:text-gray-400 mb-8">Get MyFinance UI running in minutes</p>
  
  <Steps>
    <Step title="Install dependencies">
      Clone the repository and install dependencies using pnpm.
      
      ```bash
      git clone https://github.com/EnocMag/MyFinance.UI.git
      cd MyFinance.UI
      pnpm install
      ```
    </Step>
    
    <Step title="Configure the API endpoint">
      Update the environment configuration to point to your MyFinance API backend.
      
      ```typescript src/environments/environment.ts
      export const env = {
        myFinanceApiUrl: 'http://localhost:5242/api'
      }
      ```
      
      <Note>Make sure your backend API is running and accessible at the configured URL.</Note>
    </Step>
    
    <Step title="Start the development server">
      Run the Angular development server.
      
      ```bash
      pnpm start
      ```
      
      Open your browser and navigate to `http://localhost:4200/`. The application will automatically reload when you modify source files.
    </Step>
    
    <Step title="Login and start tracking">
      Use your credentials to log in and start managing your finances. Create categories for your income and expenses, then add transactions to track your financial activity.
    </Step>
  </Steps>
</div>
