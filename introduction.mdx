## What is MyFinance UI?

MyFinance UI is a modern, Angular 21-based personal finance management application that helps you track your income, expenses, and generate comprehensive financial reports. Built with the latest web technologies, it provides an intuitive interface for managing your financial data.

The application connects to a backend API to securely store and manage your financial transactions, categories, and generate insightful reports to help you understand your spending patterns.

### Additional Capabilities

- **Date Picker Integration**: Select transaction dates easily with the flatpickr date picker
- **Real-time Form Validation**: Instant feedback on form inputs using Angular's reactive forms
- **Toast Notifications**: User-friendly success and error messages for all actions
- **Confirmation Dialogs**: Prevent accidental deletions with confirmation prompts
- **Docker Deployment**: Production-ready Docker configuration for easy deployment

## Architecture Overview

MyFinance UI is built with a modern Angular architecture:

```
src/app/
├── components/          # UI components (forms, charts, layout)
├── services/           # Business logic and API communication
├── models/             # TypeScript interfaces and data models
├── guards/             # Route guards for authentication
├── interceptors/       # HTTP interceptors for auth tokens
└── directives/         # Custom directives (date picker)
```

### Technology Stack

- **Angular 21**: Latest version of the Angular framework with standalone components
- **TypeScript 5.9**: Type-safe development experience
- **TailwindCSS 4**: Utility-first CSS framework for styling
- **Highcharts**: Interactive charts for financial visualizations
- **Flatpickr**: Lightweight date picker component
- **RxJS**: Reactive programming with observables

### Core Services

The application is organized around three main services:

- **AuthService**: Handles user authentication, login, logout, and token management (src/app/services/Auth-service.ts:1)
- **TransactionService**: Manages all transaction operations including CRUD and monthly reports
- **CategoryService**: Handles category management with type-based filtering

## API Backend

MyFinance UI requires a backend API server to function. By default, the application expects the API to be available at:

```typescript
http://localhost:5242/api
```

You can configure this endpoint in the environment configuration file at `src/environments/environment.ts`.

<Note>
  Make sure your backend API is running before starting the development server. The application uses JWT authentication and will not function without a properly configured backend.
</Note>
