## Overview

The `ReportChart` component provides interactive area charts for visualizing monthly income and expense data. It uses Highcharts library to render responsive, animated charts showing financial trends over a 12-month period.

**Location**: `~/workspace/source/src/app/components/report-chart/report-chart.ts:15`

## Data Loading Methods

### loadReportData

Fetches and processes report data for both income and expense:

```typescript
private loadReportData(): void {
  forkJoin({
    income: this.transactionService.getMonthlyReport('Income', this.currentYear - 1),
    expense: this.transactionService.getMonthlyReport('Expense', this.currentYear - 1)
  }).subscribe({
    next: ({ income, expense }) => {
      const incomeData = this.mapReportToChartData(income.data);
      const expenseData = this.mapReportToChartData(expense.data);

      this.chartOptions.series = [
        {
          type: 'area',
          name: 'Income',
          color: '#5ccb5f',
          data: incomeData
        },
        {
          type: 'area',
          name: 'Expense',
          color: '#f00',
          data: expenseData
        }
      ];

      this.updateFlag = true;
    },
    error: (error) => {
      console.error('Error loading report data:', error);
    }
  });
}
```

**Behavior**:
1. Uses `forkJoin` to fetch income and expense reports in parallel
2. Fetches data for previous year (`currentYear - 1`)
3. Maps report data to chart-compatible format
4. Updates chart series with processed data
5. Sets `updateFlag` to trigger chart re-render
6. Logs errors to console

**Note**: Currently fetches data for `currentYear - 1` instead of `currentYear`.

### mapReportToChartData

Transforms report data into chart-ready format:

```typescript
private mapReportToChartData(reports: Report[]): number[] {
  const monthlyData = new Array(12).fill(0);
  
  reports.forEach(report => {
    if (report.month >= 1 && report.month <= 12) {
      monthlyData[report.month - 1] = report.total;
    }
  });

  return monthlyData;
}
```

**Parameters**:
- `reports`: Array of `Report` objects from the API

**Returns**:
- `number[]`: Array of 12 numbers representing monthly totals

**Behavior**:
1. Creates array of 12 zeros (one per month)
2. Iterates through report data
3. Validates month is between 1-12
4. Maps report total to correct array index (month - 1)
5. Returns complete 12-month array

**Index Mapping**:
- Month 1 (January) → Index 0
- Month 2 (February) → Index 1
- ...
- Month 12 (December) → Index 11

## Data Visualization

### Chart Type: Area Chart

Area charts are chosen for:
- **Trend Visualization**: Shows changes over time
- **Comparison**: Easy to compare income vs expense
- **Volume Indication**: Filled area emphasizes magnitude

### Color Scheme

- **Income**: `#5ccb5f` (Green) - Represents positive cash flow
- **Expense**: `#f00` (Red) - Represents spending

### Data Flow

```
ngOnInit()
  ↓

  ↓

  ↓
mapReportToChartData() × 2
  ↓

  ↓
updateFlag = true
  ↓
Highcharts re-renders
```

## Performance Considerations

1. **Parallel Loading**: Uses `forkJoin` to fetch income and expense data simultaneously
2. **Change Detection**: Standalone component with default change detection
3. **Disabled Markers**: Improves rendering performance for data-heavy charts
